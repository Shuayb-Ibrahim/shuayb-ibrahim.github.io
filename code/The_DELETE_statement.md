<u>The DELETE-FROM-WHERE statement</u>
```sql
-- Deletes a record
DELETE FROM table_name
WHERE condition;
```

<u>The DELETE-FROM statement</u>
```sql
-- Deletes all the records in a table (virtually identical to truncate) 
DELETE FROM table_name;
```
