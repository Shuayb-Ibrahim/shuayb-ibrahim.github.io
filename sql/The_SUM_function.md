The `SUM()` function returns the total sum of a numeric column. The sum function only works with numeric data.

<u>The syntax for the `SUM()` function in [[The SELECT statement]] is:</u>

```sql
SELECT SUM(column_name)
FROM table_name
WHERE condition
```
