<u>The syntax for deleting a database:</u>
```sql
	-- Deleting a database 
	DROP DATABASE IF EXISTS database_name;
```

<u>The syntax for deleting a table:</u>
```sql
	-- Deleting a table 
	DROP TABLE IF EXISTS table_name;	
```
